## Installation and Running  
### 1. Download the requirement:
Download node.js  
```bash
pip install pandas  
pip install openpyxl
```  
### 2. Activate a virtual environment for django__  
```bash
cd C:\
django-env\Scripts\activate.bat   
```
### 3. ~~Start Genshin Impact~~
```bash
cd \www-genshin
npm i
npm start
```
```bash
cd \django_src
python manage.py runserver   
```
