## Installation and Running  
1. Download the requirement:  
   pip install pandas  
   pip install openpyxl  
3. Activate a virtual environment for django  
   cd C:/  
   django-env\Scripts\activate.bat   
4. ~~Start Genshin Impact~~    
   python manage.py runserver
